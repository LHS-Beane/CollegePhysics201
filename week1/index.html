    üìò 2. Position, Velocity, and Acceleration¶

    We use:

    \[ v(t)=\frac{dx}{dt} \qquad a(t)=\frac{dv}{dt} \]

    üß© Concept ‚Üí Example ‚Üí Try It¶

    Position \(x(t)\), velocity \(v(t)\), and acceleration \(a(t)\) are linked through calculus.

    Given: [ x(t) = t^3 - 4t + 2 ]

    \[ v(t) = 3t^2 - 4 \]
    \[ a(t) = 6t \]

    Find \(v(t)\) and \(a(t)\) if: [ x(t)=5t^2 - 3t + 8 ]

    Show Solution

    [ v(t)=10t-3 ] [ a(t)=10 ]

    üìò 3. Integrals in Kinematics¶

    [ v(t)=v_0 + at ] [ x(t)=x_0 + v_0 t + \frac{1}{2}at^2 ]

    Example: [ a(t)=4t ] [ v(t)=2t^2 + C ]

    [ a=g=9.8\ \text{m/s}^2 ] [ v(t)=gt + v_0 ] [ x(t)=x_0 + v_0 t + \frac{1}{2}gt^2 ]

    üìò 4. Graph Interpretation¶

    Graphs are essential.

    How to read motion graphs (click to expand)
    • Position slope ‚Üí velocity
    • Velocity slope ‚Üí acceleration
    • Area under velocity curve ‚Üí displacement

    üéì 5. Worked Examples¶

    Example 1 ‚Äî From x(t) to v(t) and a(t)

    [ x(t)=4t^3 - 2t ] [ v(t)=12t^2 - 2 ] [ a(t)=24t ]

    Example 2 ‚Äî Dropped Object

    A ball is dropped from height 20 m.

    Velocity after 2 s: [ v=gt=9.8(2)=19.6\ \text{m/s} ]

    Position: [ x=20 - \frac{1}{2}(9.8)(2^2)=0.4\ \text{m} ]

    Example 3 ‚Äî Integrating Non-Constant Acceleration
    \[ a(t)=6t,\quad v(0)=3 \]
    \[ v(t)=3t^2 + 3 \]
    \[ x(t)=t^3 + 3t \]

    üìù 6. Practice Problems¶

    Practice Problem 1

    Given: [ x(t)=2t^3 - 5t^2 + 7 ] Find \(v(t)\) and \(a(t)\).

    Show Answer

    [ v(t)=6t^2 - 10t ] [ a(t)=12t - 10 ]

    Practice Problem 2

    A car starts from rest and accelerates at 4 m/s¬≤ for 5 seconds.

    Show Answer
    \[ x=\frac{1}{2}at^2 = \frac{1}{2}(4)(25)=50\ \text{m} \]
    Practice Problem 3

    A ball is thrown upward at 12 m/s. Find its max height.

    Show Answer
    \[ h=\frac{v_0^2}{2g}=\frac{144}{19.6}=7.35\ \text{m} \]
    Practice Problem 4

    Convert 45 mph to m/s.

    Show Answer
    \[ 45\ \text{mph}=20.1\ \text{m/s} \]
    Practice Problem 5

    If \(v(t)=6t\) and \(x(0)=0\), find \(x(t)\).

    Show Answer
    \[ x(t)=3t^2 \]
