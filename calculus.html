Calculus Refresher for Physics 201

This page connects derivatives and integrals directly to motion, force, energy, and rotation. It’s not a full calculus course — it’s the toolbox you actually need in this class.

I. Derivatives in 1D Motion

At the core of kinematics: \[ v(t) = \frac{dx}{dt}, \qquad a(t) = \frac{dv}{dt} = \frac{d^2x}{dt^2}. \] You should be comfortable taking first and second derivatives of simple functions of \(t\).

Example 1 – Polynomial Position

\( x(t) = 3t^2 - 4t + 1 \) (m).

Then \[ v(t) = \frac{dx}{dt} = 6t - 4, \quad a(t) = \frac{dv}{dt} = 6. \]

Constant \(a\) = straight line \(v\) vs \(t\), parabola \(x\) vs \(t\).

Example 2 – Sinusoidal Motion

\( x(t) = A\cos(\omega t) \).

\[ v(t) = \frac{dx}{dt} = -A\omega\sin(\omega t),\\ a(t) = \frac{dv}{dt} = -A\omega^2\cos(\omega t) = -\omega^2 x(t). \]

This is exactly the structure of SHM.

Typical derivatives you need: powers \(t^n\), exponentials \(e^{kt}\), sin/cos, and sums/products.

II. Integrals in 1D Motion

Integrals reverse derivatives. In motion: \[ x(t) = x(t_0) + \int_{t_0}^{t} v(t')\,dt', \quad v(t) = v(t_0) + \int_{t_0}^{t} a(t')\,dt'. \] Graphically, the integral is the area under the curve.

Example 3 – Velocity given, find displacement

A particle has velocity \(v(t) = 4t\) (m/s). How far does it move from \(t=0\) to \(t=3\) s?

Show solution
\[ \Delta x = \int_{0}^{3} 4t\,dt = 4\left[\frac{t^2}{2}\right]_0^3 = 2(9) = 18\ \text{m}. \]

Example 4 – Acceleration given, find velocity

\( a(t) = 6t \), and \( v(0) = 2\,\text{m/s} \).

Show solution
\[ v(t) = v(0) + \int_0^t 6t'\,dt' = 2 + 6\left[\frac{t'^2}{2}\right]_0^t = 2 + 3t^2. \]

III. Vectors & Component-Wise Derivatives

In 2D or 3D, we differentiate components: \[ \vec r(t) = x(t)\hat i + y(t)\hat j (+ z(t)\hat k), \] \[ \vec v(t) = \frac{d\vec r}{dt} = \frac{dx}{dt}\hat i + \frac{dy}{dt}\hat j (+ \frac{dz}{dt}\hat k), \] \[ \vec a(t) = \frac{d\vec v}{dt} = \frac{d^2x}{dt^2}\hat i + \frac{d^2y}{dt^2}\hat j (+ \dots). \]

Example 5 – Projectile in vector form

\[ \vec r(t) = (v_0\cos\theta_0\, t)\,\hat i + (y_0 + v_0\sin\theta_0\, t - \tfrac12 g t^2)\,\hat j. \]

Then \[ \vec v(t) = \frac{d\vec r}{dt} = v_0\cos\theta_0\,\hat i + (v_0\sin\theta_0 - gt)\,\hat j, \] \[ \vec a(t) = \frac{d\vec v}{dt} = 0\,\hat i - g\,\hat j. \]

Same component-wise derivative rules as in single-variable calculus, but done for each component.

IV. Common Physics Integrals

A few integrals show up repeatedly:

  • Work from a variable force: \(W = \displaystyle\int F(x)\,dx\).
  • Rotational work: \(W = \displaystyle\int \tau(\theta)\,d\theta\).
  • Moment of inertia (concept): \(I = \displaystyle\int r^2\,dm\).
  • Energy in SHM from integrating forces/accelerations.

Example 6 – Work from a spring force

Spring force: \(F(x) = -kx\). Work from \(x=0\) to \(x=X\):

Show solution
\[ W = \int_0^{X} (-kx)\,dx = -k\left[\frac{x^2}{2}\right]_0^{X} = -\frac12 kX^2. \] This equals \(-\Delta U_s\) where \(U_s = \tfrac12 kx^2\).

Example 7 – Moment of inertia (conceptual)

Thin rod of length \(L\), mass \(M\), about its center: \[ I = \int r^2\,dm. \]

With uniform density, the result is \[ I = \frac{1}{12}ML^2. \] You don't need the full derivation here, but the integral form is what matters conceptually.

V. Common Calculus Mistakes in Physics

  • Forgetting that derivatives/integrals keep units (check units!).
  • Dropping the constant of integration when you actually need initial conditions.
  • Mixing up what is a function of time vs. a constant (treating constants like variables).
  • Using kinematics formulas in situations where acceleration is not constant.
  • Sign errors when integrating or differentiating vector components.

VI. Practice Problems (with Solutions)

Try these without looking at the solutions first. Each one is designed to match something that appears in the 8-week course.

Problem 1 – Velocity & Acceleration from Position

A particle moves along the \(x\)-axis with position \[ x(t) = 2t^3 - 5t^2 + 4 \quad (\text{meters}). \] Find \(v(t)\) and \(a(t)\).

Solution:

\[ v(t) = \frac{dx}{dt} = 6t^2 - 10t, \] \[ a(t) = \frac{dv}{dt} = 12t - 10. \]

Problem 2 – Displacement from Velocity Graph

Velocity is \(v(t) = 3\) m/s for \(0 \le t \le 2\) s, then \(v(t) = 6\) m/s for \(2 \le t \le 4\) s. If \(x(0) = 0\), find \(x(4)\).

Solution:

Displacement from 0 to 2: \[ \Delta x_1 = \int_0^2 3\,dt = 3 \cdot 2 = 6\ \text{m}. \] From 2 to 4: \[ \Delta x_2 = \int_2^4 6\,dt = 6 \cdot 2 = 12\ \text{m}. \] Total \(x(4) = 6 + 12 = 18\ \text{m}.\)

Problem 3 – Work from a Linearly Changing Force

A force along the x-axis is \(F(x) = 10x\) N. How much work is done as the object moves from \(x=0\) m to \(x=2\) m?

Solution:

\[ W = \int_0^2 10x\,dx = 10\left[\frac{x^2}{2}\right]_0^2 = 5(4) = 20\ \text{J}. \]

Problem 4 – SHM from Differential Equation

A mass–spring system satisfies \[ \frac{d^2x}{dt^2} + \omega^2 x = 0. \] Show that \(x(t) = A\cos(\omega t) + B\sin(\omega t)\) is a solution.

Solution:

Differentiate twice:

\[ x(t) = A\cos(\omega t) + B\sin(\omega t), \] \[ x'(t) = -A\omega\sin(\omega t) + B\omega\cos(\omega t), \] \[ x''(t) = -A\omega^2\cos(\omega t) - B\omega^2\sin(\omega t) = -\omega^2 x(t). \] So \[ x'' + \omega^2 x = -\omega^2 x + \omega^2 x = 0, \] so it satisfies the equation.

Problem 5 – Velocity Components for Circular Motion

A particle moves in a circle of radius \(R\) with angular position \(\theta(t) = \omega t\). Position vector: \[ \vec r(t) = R\cos(\omega t)\,\hat i + R\sin(\omega t)\,\hat j. \] Find \(\vec v(t)\) and \(\vec a(t)\).

Solution:

\[ \vec v(t) = \frac{d\vec r}{dt} = -R\omega\sin(\omega t)\,\hat i + R\omega\cos(\omega t)\,\hat j, \] with magnitude \(v = R\omega\).

\[ \vec a(t) = \frac{d\vec v}{dt} = -R\omega^2\cos(\omega t)\,\hat i - R\omega^2\sin(\omega t)\,\hat j = -\omega^2 \vec r(t), \] so acceleration is radial inward with magnitude \(a_c = R\omega^2\).
