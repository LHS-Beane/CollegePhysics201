Units, Dimensions & 1D Kinematics with Calculus

This week you will connect derivatives and integrals with physical motion. We start with units and dimensions, then describe position, velocity, and acceleration as functions of time in one dimension, including constant and non-constant acceleration.

Learning Goals

  • Express position $x(t)$, velocity $v(t)$, and acceleration $a(t)$ as functions of time.
  • Interpret velocity as the derivative of position: $v(t) = \\dfrac{dx}{dt}$.
  • Interpret acceleration as the derivative of velocity: $a(t) = \\dfrac{dv}{dt}$.
  • Use integrals to recover velocity and position from acceleration.
  • Apply kinematic relationships to motion with constant acceleration (including free fall).

1. Units, Dimensions, and Coordinate Systems

We work in the SI system: meters (m) for length, seconds (s) for time, and kilograms (kg) for mass. Dimensional analysis helps check if an equation is physically reasonable.

Example: In $x = x_0 + v_0 t + \\tfrac{1}{2} a t^2$, each term must have dimensions of length $[L]$:

  • $[x_0] = [L]$
  • $[v_0 t] = [L/T] [T] = [L]$
  • $[a t^2] = [L/T^2] [T^2] = [L]$

2. Position, Velocity, and Acceleration

In one dimension, we describe motion along a line with a function $x(t)$.

Definitions (Calculus-Based):

$v(t) = \\dfrac{dx}{dt}$ — instantaneous velocity is the time derivative of position.

$a(t) = \\dfrac{dv}{dt} = \\dfrac{d^2 x}{dt^2}$ — instantaneous acceleration is the derivative of velocity, or the second derivative of position.

If you know $a(t)$ and an initial velocity $v(t_0)$, you can integrate to find velocity: $$v(t) = v(t_0) + \\int_{t_0}^{t} a(t')\\, dt'$$ and then integrate again to find position: $$x(t) = x(t_0) + \\int_{t_0}^{t} v(t')\\, dt'$$

3. Motion with Constant Acceleration

A very important special case is when acceleration is constant: $a(t) = a_0$.

Starting from $a = \\dfrac{dv}{dt} = a_0$, integrate:

  • $v(t) = v_0 + a_0 (t - t_0)$
  • $x(t) = x_0 + v_0 (t - t_0) + \\tfrac{1}{2} a_0 (t - t_0)^2$

Another useful relationship (eliminating time) is obtained by combining definitions:

$$v^2 = v_0^2 + 2 a_0 (x - x_0)$$

In free fall near Earth’s surface, we often take $a_0 = -g \\approx -9.8\\,\\text{m/s}^2$.

Example 1: Constant Acceleration from Calculus

A particle moves along the $x$-axis with constant acceleration $a = 3.0\\,\\text{m/s}^2$. At $t=0$, the particle is at $x_0 = 2.0\\,\\text{m}$ with velocity $v_0 = 1.0\\,\\text{m/s}$.

  1. Find $v(t)$ using calculus.
  2. Find $x(t)$ using calculus.
  3. Determine $x$ and $v$ at $t = 4.0\\,\\text{s}$.
Show Solution

(1) Velocity: Start from $a = \\dfrac{dv}{dt} = 3.0$.

Integrate with respect to $t$:

$$v(t) = \\int a\\, dt = \\int 3.0\\, dt = 3.0 t + C$$

Use $v(0) = v_0 = 1.0$ to find $C$:

$1.0 = 3.0(0) + C \\Rightarrow C = 1.0$

So: $$v(t) = 3.0 t + 1.0\\,\\text{(m/s)}$$

(2) Position: $v = \\dfrac{dx}{dt} = 3.0 t + 1.0$.

Integrate again:

$$x(t) = \\int v(t)\\, dt = \\int (3.0 t + 1.0)\\, dt = 1.5 t^2 + t + C'$$

Use $x(0) = x_0 = 2.0$:

$2.0 = 1.5(0)^2 + 0 + C' \\Rightarrow C' = 2.0$

So: $$x(t) = 1.5 t^2 + t + 2.0\\,\\text{(m)}$$

(3) Evaluate at $t = 4.0$ s:

$v(4.0) = 3.0(4.0) + 1.0 = 13.0\\,\\text{m/s}$.

$x(4.0) = 1.5(4.0)^2 + 4.0 + 2.0 = 1.5(16) + 6 = 24 + 6 = 30\\,\\text{m}$.

Example 2: Position from a Time-Dependent Acceleration

Suppose a particle has acceleration $a(t) = 4t\\,\\text{(m/s}^2\\text{)}$ along the $x$-axis. At $t = 0$, $v_0 = 0$ and $x_0 = 0$.

  1. Find $v(t)$ using integration.
  2. Find $x(t)$.
Show Solution

(1) Velocity:

$a(t) = \\dfrac{dv}{dt} = 4t$

$$v(t) = \\int 4t\\, dt = 2t^2 + C$$

Use $v(0) = 0$ ⇒ $C=0$.

So: $v(t) = 2t^2$.

(2) Position:

$v(t) = \\dfrac{dx}{dt} = 2t^2$

$$x(t) = \\int 2t^2\\, dt = \\frac{2}{3} t^3 + C'$$

Use $x(0) = 0$ ⇒ $C' = 0$.

So: $x(t) = \\dfrac{2}{3} t^3$.

Problem 1 – Free Fall from Rest

A rock is dropped from rest from a height of $h = 20\\,\\text{m}$ above the ground. Take downward as positive and use $g = 9.8\\,\\text{m/s}^2$.

  1. Write expressions for $v(t)$ and $x(t)$ using calculus.
  2. Find the time it takes to hit the ground.
  3. Find the speed just before impact.

Solution (outline):

$a = g = 9.8$ so $v(t) = gt + v_0 = 9.8 t$ with $v_0 = 0$.

$x(t) = x_0 + \\int_0^t v(t') dt' = h + \\int_0^t 9.8 t' dt' = 20 + 4.9 t^2$.

Set ground at $x = 0$. Then $0 = 20 + 4.9 t^2$ ⇒ $t = \\sqrt{-20/4.9}$ but sign convention will matter.

You can instead take upward positive with $a = -9.8\\,\\text{m/s}^2$; the standard solution should give $t \\approx 2.0\\,\\text{s}$ and $v \\approx 19.8\\,\\text{m/s}$ downward. Work it carefully with consistent sign conventions.

Problem 2 – Velocity from a Given Acceleration

A particle moves along the $x$-axis with acceleration $a(t) = -6t$. At $t = 0$, we have $v_0 = 5.0\\,\\text{m/s}$ and $x_0 = 0$.

  1. Find $v(t)$ using integration.
  2. Find $x(t)$.
  3. At what time does the particle momentarily come to rest?

Solution (outline):

$v(t) = \\int a(t)dt = \\int (-6t) dt = -3t^2 + C$.

Use $v(0) = 5$ ⇒ $C = 5$ ⇒ $v(t) = 5 - 3t^2$.

$x(t) = \\int v(t) dt = \\int (5 - 3t^2) dt = 5t - t^3 + C'$; use $x(0) = 0$ ⇒ $C' = 0$.

Set $v(t) = 0$: $5 - 3t^2 = 0$ ⇒ $t = \\sqrt{5/3} \\approx 1.29\\,\\text{s}$.

Problem 3 – Graph Interpretation

Suppose the velocity of a particle is given by the graph of $v(t)$ vs $t$ (not shown here). Describe in words how you would compute the displacement between $t = 1\\,\\text{s}$ and $t = 4\\,\\text{s}$ using calculus, and how this relates to the area under the curve.

Solution (conceptual):

The displacement is $$\\Delta x = x(4) - x(1) = \\int_{1}^{4} v(t)\\, dt.$$ Geometrically, this is the signed area under the $v(t)$ curve between $t = 1$ and $t = 4$.
