Week 1 • Sessions 1–2

Units, Vectors, and 1D Kinematics with Calculus

This module introduces the language of physics: units, dimensions, vectors, and the calculus-based description of motion in one dimension. The goal is to tie what you know from calculus directly to real physical motion.

Learning Goals

  • Use and interpret SI units, prefixes, and dimensions for physical quantities.
  • Apply dimensional analysis to check equations and infer relationships.
  • Distinguish clearly between scalars and vectors, and represent vectors graphically and in components.
  • Relate position, velocity, and acceleration using derivatives and integrals.
  • Derive and apply the constant-acceleration kinematic equations, including free-fall motion.
  • Interpret and connect \(x(t)\), \(v(t)\), and \(a(t)\) graphs.

How to Use This Module

  • Read the Notes section carefully, pausing to work through each example.
  • Use the diagrams to connect formulas to geometry and graphs.
  • Work through the Worked Examples start to finish.
  • Attempt the Practice Problems without looking at solutions; then check your reasoning.
  • Use the Interactive Panel to experiment with constant-acceleration motion.

1. Physical Quantities, Units, and Dimensions

Physics describes the world using measurable quantities. Every quantity has:

  • a dimension (type of quantity, like length or time), and
  • a unit (standard used to measure that dimension).
Quantity Dimension SI Unit Symbol
Length [L] meter m
Mass [M] kilogram kg
Time [T] second s
Velocity [L/T] meter/second m/s

Many everyday prefixes are powers of ten:

Prefix Symbol Factor
milli m \(10^{-3}\)
centi c \(10^{-2}\)
kilo k \(10^{3}\)
mega M \(10^{6}\)

2. Dimensional Analysis

Dimensional analysis uses the basic dimensions of a quantity (such as [L], [M], [T]) to check whether equations are consistent and to guess how different quantities might relate.

Example: Check the kinematic equation \[ v = v_0 + a t. \]

  • Left side: \(v\) has dimensions [L/T].
  • Right side: \(v_0\) has [L/T], and \(a t\) has \([L/T^2] [T] = [L/T]\).

Both sides have the same dimensions, so the equation is dimensionally consistent.

Example of an inconsistent formula:

Suppose someone proposes \(x = v^2 t\). Dimensions: \[ [x] = [L], \qquad [v^2 t] = (L/T)^2 \cdot T = [L^2/T]. \] These do not match, so the equation cannot be physically correct.

Using dimensions to infer a formula

Assume the period \(T\) of a simple pendulum depends only on its length \(L\) and the gravitational acceleration \(g\). Guess \(T \propto L^a g^b\). Use dimensions:

\([T] = [L]^a [L/T^2]^b = [L]^{a+b} [T]^{-2b}\). Matching exponents: time gives \(1 = -2b\Rightarrow b=-\tfrac12\); length gives \(0=a+b\Rightarrow a=\tfrac12\). So \(T \propto \sqrt{L/g}\).

3. Scalars and Vectors

A scalar has magnitude only (e.g., mass, temperature, time). A vector has both magnitude and direction (e.g., displacement, velocity, acceleration, force).

In two dimensions, a vector \(\vec{A}\) can be written in component form: \[ \vec{A} = A_x \hat{i} + A_y \hat{j}. \]

From the diagram, the magnitude and direction of \(\vec{A}\) are \[ |\vec{A}| = \sqrt{A_x^2 + A_y^2}, \qquad \theta = \tan^{-1}\!\left(\frac{A_y}{A_x}\right). \]

Vector addition (head-to-tail):

Algebraically, if \(\vec{R} = \vec{A} + \vec{B}\), then \[ R_x = A_x + B_x, \qquad R_y = A_y + B_y. \]

4. Calculus-Based Description of Motion in 1D

In one dimension, motion is along a single axis (we will call it the \(x\)-axis). We describe position as a function of time: \(x(t)\).

The calculus definitions:

Velocity: \(v(t) = \dfrac{dx}{dt}\) (instantaneous rate of change of position).

Acceleration: \(a(t) = \dfrac{dv}{dt} = \dfrac{d^2 x}{dt^2}\) (instantaneous rate of change of velocity).

To reverse the process, we integrate:

If we know \(a(t)\) and the velocity at some initial time \(t_0\), \[ v(t) = v(t_0) + \int_{t_0}^{t} a(t')\, dt'. \]

If we then know \(v(t)\) and the position at \(t_0\), \[ x(t) = x(t_0) + \int_{t_0}^{t} v(t')\, dt'. \]

On an \(x\)-versus-\(t\) graph, the slope at each point is the instantaneous velocity. If the graph curves, the slope is changing, which means the acceleration is nonzero.

5. Motion with Constant Acceleration

A very important special case is when acceleration is constant: \(a(t) = a\).

Deriving the kinematic equations

Start from the definition \(a = \dfrac{dv}{dt} = a\).

Integrate: \[ v(t) = v_0 + a t \] where \(v_0 = v(0)\).

Now use \(v = \dfrac{dx}{dt} = v_0 + a t\), and integrate again: \[ x(t) = x_0 + v_0 t + \tfrac12 a t^2, \] where \(x_0 = x(0)\).

Eliminating time gives the third kinematic relationship: \[ v^2 = v_0^2 + 2a(x - x_0). \]

On a \(v\)-versus-\(t\) graph, the area under the curve between \(t_1\) and \(t_2\) is the displacement: \[ \Delta x = \int_{t_1}^{t_2} v(t)\, dt. \]

6. Free Fall as Constant Acceleration

Near Earth’s surface, an object in free fall (neglecting air resistance) has an approximately constant downward acceleration of magnitude \(g \approx 9.8\,\text{m/s}^2\).

If we choose upward as positive, then \[ a = -g. \]

The motion is described by \[ v(t) = v_0 - g t, \qquad x(t) = x_0 + v_0 t - \tfrac12 g t^2. \]

Acceleration is (approximately) constant in free fall.

Positions at equal time intervals get farther apart as the object speeds up.

7. Summary of Key Relationships

  • Use SI units (m, kg, s) and dimensional analysis to check equations.
  • Vectors have magnitude and direction; resolve them into components using trig.
  • Velocity is the derivative of position; acceleration is the derivative of velocity.
  • Integrals of \(a(t)\) and \(v(t)\) give you \(v(t)\) and \(x(t)\), respectively, up to constants of integration set by initial conditions.
  • For constant acceleration: \[ v = v_0 + at, \quad x = x_0 + v_0 t + \tfrac12 a t^2, \quad v^2 = v_0^2 + 2a(x - x_0). \]
  • Free fall is constant acceleration with \(a = -g\) when up is positive.
  • The slope of \(x(t)\) is \(v(t)\); the slope of \(v(t)\) is \(a(t)\); areas under curves correspond to changes in position or velocity.

Example 1: Constant Acceleration from Calculus

A particle moves along the \(x\)-axis with constant acceleration \(a = 3.0\,\text{m/s}^2\). At \(t = 0\), the particle is at \(x_0 = 2.0\,\text{m}\) with velocity \(v_0 = 1.0\,\text{m/s}\).

  1. Find \(v(t)\) using calculus.
  2. Find \(x(t)\) using calculus.
  3. Determine \(x\) and \(v\) at \(t = 4.0\,\text{s}\).
Show Solution

(1) Velocity: Start from \(a = \dfrac{dv}{dt} = 3.0\).

Integrate: \[ v(t) = \int 3.0\, dt = 3.0 t + C. \]

Use \(v(0) = v_0 = 1.0\) \(\Rightarrow C = 1.0\). So \[ v(t) = 3.0 t + 1.0\,\text{m/s}. \]

(2) Position: \(v = \dfrac{dx}{dt} = 3.0 t + 1.0\).

\[ x(t) = \int (3.0 t + 1.0)\, dt = 1.5 t^2 + t + C'. \]

Use \(x(0) = x_0 = 2.0\) \(\Rightarrow C' = 2.0\). So \[ x(t) = 1.5 t^2 + t + 2.0\,\text{m}. \]

(3) At \(t = 4.0\,\text{s}\):

\(v(4.0) = 3.0(4.0) + 1.0 = 13.0\,\text{m/s}.\)

\(x(4.0) = 1.5(4.0)^2 + 4.0 + 2.0 = 1.5(16) + 6 = 24 + 6 = 30\,\text{m}.\)

Example 2: Time-Dependent Acceleration

A particle moves along the \(x\)-axis with acceleration \(a(t) = 4t\,\text{(m/s}^2\text{)}\). At \(t = 0\), \(v_0 = 0\) and \(x_0 = 0\).

  1. Find \(v(t)\) by integration.
  2. Find \(x(t)\).
Show Solution

(1) Velocity: \(a(t) = \dfrac{dv}{dt} = 4t\).

\[ v(t) = \int 4t\, dt = 2t^2 + C. \]

Use \(v(0) = 0\) \(\Rightarrow C = 0\). So \(v(t) = 2t^2.\)

(2) Position: \(v(t) = \dfrac{dx}{dt} = 2t^2\).

\[ x(t) = \int 2t^2\, dt = \tfrac{2}{3} t^3 + C'. \]

Use \(x(0) = 0\) \(\Rightarrow C' = 0\). So \(x(t) = \tfrac{2}{3} t^3.\)

Practice Set – Week 1

Try each problem fully before expanding the solution outline. Focus on setting up correct expressions using units, vectors, and calculus.

Problem 1 – Dimensional Analysis

A student claims the kinetic energy of a particle can be written as \[ K = m v^3. \]

  1. Use dimensions to test whether this could be correct.
  2. Compare with the standard expression \(K = \tfrac12 m v^2\).

Solution (outline):

For \(K = m v^3\): \([K] = [M] [L/T]^3 = [M L^3/T^3]\), but energy in SI has dimensions \([M L^2/T^2]\). The proposed formula is dimensionally inconsistent.

Problem 2 – Velocity from a Given Acceleration

A particle has acceleration \(a(t) = -6t\) along the \(x\)-axis. At \(t = 0\), \(v_0 = 5.0\,\text{m/s}\) and \(x_0 = 0\).

  1. Find \(v(t)\) using integration.
  2. Find \(x(t)\).
  3. At what time does the particle momentarily come to rest?

Solution (outline):

\(v(t) = \int (-6t)\, dt = -3t^2 + C\). Use \(v(0) = 5\Rightarrow C = 5\), so \(v(t) = 5 - 3t^2.\)

\(x(t) = \int (5 - 3t^2)\, dt = 5t - t^3 + C'\). With \(x(0)=0\Rightarrow C' = 0\).

Set \(v(t)=0\): \(5 - 3t^2 = 0 \Rightarrow t = \sqrt{5/3} \approx 1.29\,\text{s}.\)

Problem 3 – Free-Fall from a Height

A rock is dropped from rest from a height of \(h = 20\,\text{m}\) above the ground. Take upward as positive and use \(g = 9.8\,\text{m/s}^2\).

  1. Write \(x(t)\) and \(v(t)\) for the rock’s motion.
  2. Find the time when the rock hits the ground.
  3. Find the speed just before impact.

Solution (outline):

With upward positive, \(a = -g\), \(v_0 = 0\), \(x_0 = h\): \[ v(t) = -g t,\quad x(t) = h - \tfrac12 g t^2. \]

Impact when \(x(t) = 0\): \(0 = h - \tfrac12 g t^2 \Rightarrow t = \sqrt{\frac{2h}{g}} \approx 2.0\,\text{s}.\)

\(v(t) = -g t \approx -19.6\,\text{m/s}\); speed is \(19.6\,\text{m/s}\) downward.
