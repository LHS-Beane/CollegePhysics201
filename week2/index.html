    üìò 1. Introduction to Vectors¶

    1.1 What is a Vector?

    A vector has both:

    • magnitude (how much)
    • direction (which way)

    Examples: - velocity
    - acceleration
    - displacement
    - force

    Scalar ‚â† vector
    - a scalar has magnitude only
    - example: mass, temperature, energy

    1.2 Vector Notation

    Common notations:

    \[ \vec{A},\quad \mathbf{A},\quad \langle A_x, A_y \rangle \]

    Components: [ \vec{A} = \langle A_x,\ A_y \rangle ]

    Magnitude: [ |\vec{A}| = \sqrt{A_x^2 + A_y^2} ]

    Unit vector form: [ \vec{A} = A_x \hat{i} + A_y \hat{j} ]

    Deep Explanation ‚Äî Why Physics Uses Components

    Real-world motion rarely happens in a straight line.

    By breaking motion into: [ x\text{-direction} \quad \text{and} \quad y\text{-direction} ] we can:

    • treat each dimension independently
    • solve harder problems using simple 1D equations
    • combine them at the end to get full motion

    Example:
    Projectile motion has: - horizontal motion with constant velocity
    - vertical motion with constant acceleration

    Components allow us to separate these behaviors cleanly.

    üìò 2. Vector Components¶

    Any vector can be broken into x and y components:

    \[ A_x = A\cos\theta,\qquad A_y = A\sin\theta \]

    This allows you to treat motion separately in each direction.

    A 10 m vector at 30¬∞ above horizontal:

    [ A_x = 10\cos 30^\circ = 8.66 ] [ A_y = 10\sin 30^\circ = 5 ]

    A 12 m vector at 60¬∞:

    Show Solution

    [ A_x = 12\cos 60^\circ = 6 ] [ A_y = 12\sin 60^\circ = 10.39 ]

    üìò 3. Vector Addition & Subtraction¶

    Click to expand

    For vectors: [ \vec{A} = \langle A_x, A_y \rangle,\quad \vec{B} = \langle B_x, B_y \rangle ]

    Addition: [ \vec{A} + \vec{B} = \langle A_x + B_x,\ A_y + B_y \rangle ]

    Subtraction: [ \vec{A} - \vec{B} = \langle A_x - B_x,\ A_y - B_y \rangle ]

    Geometric interpretation:
    - Tip-to-tail method
    - Parallelogram rule

    üìò 4. Dot Product (Optional but powerful)¶

    Understanding the Dot Product (Why it matters)

    Definition:

    \[ \vec{A}\cdot\vec{B} = A_x B_x + A_y B_y \]

    Geometric form:

    \[ \vec{A}\cdot\vec{B} = |\vec{A}|\,|\vec{B}|\cos\theta \]

    Meaning: - Measures how much two vectors ‚Äúpoint in the same direction‚Äù
    - Used heavily in energy & work (Week 4)
    - Helps find angles between vectors

    üìò 5. 2D Kinematics Overview¶

    Motion in 2 dimensions is broken into two independent motions:

    • Horizontal (x): constant velocity
    • Vertical (y): constant acceleration
    \[ x(t)=v_{0x}t, \qquad y(t)=y_0 + v_{0y} t + \frac{1}{2}(-g)t^2 \]

    Acceleration only affects the vertical motion.

    Deep Explanation ‚Äî Why x and y Motion Are Independent

    Gravity only pulls downward.

    That means: - it changes vertical motion
    - it does NOT affect horizontal motion

    This is why a bullet dropped and a bullet fired horizontally
    hit the ground at the same time.

    üìò 6. Projectile Motion¶

    A projectile has: - horizontal motion with constant velocity
    [ x(t)=v_{0x} t ] - vertical motion with constant acceleration
    [ y(t)=y_0 + v_{0y} t - \frac{1}{2}gt^{2} ]

    A ball is launched at \(20\ \text{m/s}\) at 40¬∞.

    Components: [ v_{0x} = 20\cos 40^\circ = 15.32 ] [ v_{0y} = 20\sin 40^\circ = 12.86 ]

    Launch speed \(15\ \text{m/s}\) at 30¬∞.

    Show Solution

    [ v_{0x} = 15\cos 30^\circ = 12.99 ] [ v_{0y} = 15\sin 30^\circ = 7.5 ]

    üéì 7. Worked Examples¶

    Example 1 ‚Äî Range of a Projectile

    A ball is launched at \(25\ \text{m/s}\) at 35¬∞ from ground level.

    Components: [ v_{0x} = 25\cos 35^\circ = 20.48 ] [ v_{0y} = 25\sin 35^\circ = 14.34 ]

    Time of flight: [ t = \frac{2v_{0y}}{g} = 2.93\ \text{s} ]

    Range: [ R=v_{0x}t = 59.97\ \text{m} ]

    Example 2 ‚Äî Maximum Height

    Using: [ v_y^2 = v_{0y}^2 - 2g(y - y_0) ]

    Max height occurs when: [ v_y = 0 ]

    Solve: [ h = \frac{v_{0y}^2}{2g} ]

    Example 3 ‚Äî Relative Velocity

    A boat moves 5 m/s east in a river flowing 3 m/s north.

    \[ \vec{v} = \langle 5,\ 3 \rangle \]

    Magnitude: [ |\vec{v}| = \sqrt{5^2 + 3^2} = 5.83\ \text{m/s} ]

    üìù 8. Practice Problems¶

    1. A 12 m vector is at 25¬∞ above horizontal. Find components.
    Show Answer

    [ A_x = 12\cos 25^\circ = 10.88 ] [ A_y = 12\sin 25^\circ = 5.07 ]

    2. A projectile is launched at 18 m/s at 45¬∞.

    Find its horizontal and vertical components.

    Show Answer

    [ v_{0x} = 18\cos 45^\circ = 12.73 ] [ v_{0y} = 18\sin 45^\circ = 12.73 ]

    3. Find the range of a projectile launched at 20 m/s at 30¬∞.
    Show Answer

    [ v_{0x}=17.32,\quad v_{0y}=10 ] [ t = \frac{2v_{0y}}{g} = 2.04\ \text{s} ] [ R = v_{0x} t = 35.3\ \text{m} ]

    4. A ball is thrown horizontally at 8 m/s from 10 m height.
    Show Answer

    Time to fall: [ t = \sqrt{\frac{2h}{g}} = 1.43\ \text{s} ]

    Horizontal distance: [ x = v_x t = 11.4\ \text{m} ]

    5. A boat moves 4 m/s east in a 3 m/s north current.
    Show Answer

    [ \vec{v}=\langle 4,\ 3 \rangle ] [ |\vec{v}| = 5\ \text{m/s} ]
